# Microservices Ecommerce

Proyecto base de microservicios para un e-commerce usando Spring Boot.

## Microservicios

- `discovery-server`: servidor de descubrimiento (Eureka) para registrar y ubicar los servicios.
- `product-service`: gestiona el catalogo de productos.
- `inventory-service`: gestiona inventario y stock.
- `order-service`: gestiona pedidos y su flujo.
- `notification-service`: envio de notificaciones (correo/mensajeria).

## Requisitos minimos

- Java 21
- Maven Wrapper (`mvnw` / `mvnw.cmd`)
